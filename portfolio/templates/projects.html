Peachy Planet

View source code

www.peachyplanet.us

The idea is co-workers of a small company may work with different people on different projects. This is a way for them to follow and unfollow each other as needed to stay up to date on each other's progress etc.

This was derived from a team treehouse tutorial by Kenneth Love

It is a very basic social website built with

  • Python and Flask for the MVC web framework

  • Bcrypt was used for hashing passwords

  • wtforms was used for the forms and validation

  • Flask-login was used for authentication

  • Peewee was used as the ORM for the database.

  • HTML, CSS, and JavaScript were used for the GUI

Place Holder Image Server

View source code

2achary.pythonanywhere.com

This server can be used for serving placeholder images for any web page.

To request a placeholder image of a given width and height simply include an image with the source pointing to /placeholder/x/ on this server such as:

<img src="http://localhost:8000/image/50x50/" >

Details

This project demonstrates the ability to use Django in a single file fashion taking advantage of it's decoupled nature. For example, the django-admin startproject command was never used to create the folder structure.

Python Math Quiz

View source code

A timed math quiz in python with a CLI.

Each question is timed and once an answer is provided, there is instant feedback as to whether it was right or wrong.

Once all the questions have been answered, there is a summary of each questions correctness, the average time per question and total time of quiz.

This project demonstrates the ability to manipulate date and time objects in Python, creating a CLI, input validation, string manipulation, and clean, modular, testable code. There are unittests included in the repo.

Project Uses
  • Pyton's datetime library to record and measure testing metrics
  • Command line interface
  • Modular, testable code
  • Python's Unittest module with tests included
